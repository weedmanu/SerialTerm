# Application Layer

Ce répertoire contient les "Use Cases" (cas d'utilisation) de l'application.
Ils servent de pont applicatif entre l'interface utilisateur (`ui`) et la logique de domaine (`core`).

C'est ici que s'effectuent par exemple les résolutions de règles de configuration, la validation croisée des champs formulaires avant initialisation physique de la connexion, et les calculs propres à l'instance unifiée de l'application.
