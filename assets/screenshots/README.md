# Screenshots

Ces captures d'ecran illustrent les principales fonctionnalites de l'application.

## Fichiers publies

- SerialTerm_001.png - Connexion serie, selection du port a 115200 bauds.
- SerialTerm_002.png - Reglages serie avances : bits de donnees, parite, stop, flux.
- SerialTerm_003.png - Terminal en session active avec flux ANSI.
- SerialTerm_004.png - Outils integres et consultation des logs.

## Procedure

Avant une release :

1. Lancer l'application dans un etat representatif.
2. Capturer les vues necessaires avec gnome-screenshot ou flameshot.
3. Redimensionner si besoin a une taille stable.
4. Remplacer les fichiers publies ici en conservant les noms references par le metainfo.
5. Pousser ces captures sur la branche publiee du depot TutoElectroWeb avant de creer le tag de release.
